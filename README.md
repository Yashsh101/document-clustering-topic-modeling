# 🧠 Document Clustering & Topic Modeling

## 🚀 Overview

This project implements an unsupervised machine learning pipeline to extract meaningful topics from large text datasets. It leverages clustering and topic modeling techniques to uncover hidden patterns in unstructured data.

## 🎯 Objective

To analyze and group similar documents while identifying key themes using NLP techniques.

## 🧠 Features

* Text preprocessing (cleaning, tokenization)
* TF-IDF vectorization
* K-Means clustering
* Latent Dirichlet Allocation (LDA)
* Topic extraction and interpretation
* Dimensionality reduction using t-SNE

## 🛠️ Tech Stack

* Python
* Scikit-learn
* NLTK
* Pandas, NumPy
* Matplotlib / Seaborn
